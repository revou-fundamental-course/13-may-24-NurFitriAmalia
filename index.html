Luas Segitiga

Rumus Luas Segitiga yaitu :
L = 1/2 x a x t
Di mana :
L = Luas
a = Panjang Alas
t = Tinggi

Keliling Segitiga

Rumus Keliling Segitiga yaitu :
K = S1 + S2 + S3
Di mana :
K = Keliling
S1 = Sisi A
S2 = Sisi B
S3 = Sisi C
